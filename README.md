## Tech Stack

| Category | Technologies |
|----------|-------------|
| **Framework** | [Astro](https://astro.build) (Static Output) |
| **UI Library** | [React 19](https://react.dev) + [TypeScript](https://typescriptlang.org) |
| **Styling** | [Tailwind CSS v4](https://tailwindcss.com) + [shadcn/ui](https://ui.shadcn.com) |
| **Animations** | [Framer Motion](https://motion.dev) (Scroll-based parallax) |
| **State** | [Zustand](https://zustand-demo.pmnd.rs) (Theme + UI state) |
| **Data Fetching** | [TanStack Query](https://tanstack.com/query) |
| **Validation** | [Zod](https://zod.dev) (Type-safe form validation) |
| **Icons** | [HugeIcons](https://hugeicons.com) (`@hugeicons/react`) |
| **Forms** | [Web3Forms](https://web3forms.com) (Serverless submissions) |
| **Deployment** | [GitHub Pages](https://pages.github.com) (Static hosting) |

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org) 18+ or [Bun](https://bun.sh) 1.0+

### Installation

```bash
# Clone the repository
git clone https://github.com/FiazTechnologies/fiaztechnologies.github.io.git
cd fiaztechnologies.github.io

# Install dependencies
bun install
# or
npm install
```

### Development

```bash
# Start development server
bun run dev
# or
npm run dev
```

### Build

```bash
# Build for production
bun run build
# or
npm run build

# Preview production build
bun run preview
```

The production build outputs to the `dist/` directory, ready for deployment.

---

## Deployment

This project is configured for **GitHub Pages** deployment. The static output is generated in the `dist/` directory.

### Automatic Deployment

Push to the `main` branch and configure GitHub Pages to deploy from the `dist/` directory, or use a GitHub Actions workflow.

### Manual Deployment

```bash
bun run build
# Deploy the dist/ directory to your hosting provider
```
